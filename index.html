Tingle.js

Step by step tutorial for people who kinda know what they're doing

  1. Open a new folder and create a new file index.html
  2. Open terminal and run a command npm init
  3. Unless you want to add some information, press enter through the initialization
  4. Then, run a command npm install tingle.js --save
  5. Link tingle's js and css files to your html 
                    <link rel="stylesheet" href="node_modules/tingle.js/dist/tingle.min.css" />
                <script src="node_modules/tingle.js/dist/tingle.js"> < /script >
  6. Create a js file, for example tingle.js, and link it to your html file
  7. Import tingle by writing import tingle from "tingle.js";
  8. The pattern for creating a modal looks like this: 
             // instanciate new modal
        var modal = new tingle.modal({
            footer: true,
            stickyFooter: false,
            closeMethods: ['overlay', 'button', 'escape'],
            closeLabel: "Close",
            cssClass: ['custom-class-1'],
            onOpen: function() {
                console.log('modal open');
            },
            onClose: function() {
                console.log('modal closed');
            },
            beforeClose: function() {
            // here's goes some logic
            // e.g. save content before closing the modal
            return true; // close the modal
            return false; // nothing happens
            }
        });
                    
        // set content
        modal.setContent();
                    
        // add a button
        modal.addFooterBtn('Button label', 'tingle-btn tingle-btn--primary', function() {
            // here goes some logic
            modal.close();
        });
                    
        // add another button
        modal.addFooterBtn('Dangerous action !', 'tingle-btn tingle-btn--danger', function() {
            // here goes some logic
            modal.close();
        });
                    
        // open modal
        modal.open();
                    
        // close modal
        modal.close();
    Looks complicated? Don't worry, just paste it into your js file and we'll figure it all out.
  9. At first, find the line of code which sets content and add some text there, for example 
            // set content
        modal.setContent("<h1>This is my content</h1>");
  10. Create a button for opening the modal and add some code in js, that runs the funtion modal.open() when the button is clicked.
  11. Great! You can now see your modal. It looks pretty, but you can change some styling. I did that by creating my own css file and conecting it to the index.html after the tingle.min.css one
  12. You can grab the whole modal by your custom class (which you can create in var modal) or by class tingle.modal
  13. To style the buttons use: tingle-btn in general, tingle-modal__close for the close button, tingle-btn--primary and tingle-btn--danger for the footer buttons. In js, you can also disable the footer (by giving it the value of false) and add/remove footer buttons.
  14. That's it! Play around with the modal!
